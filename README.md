# Water Displacement Donation Website

A simple static website created to raise awareness and support for people who have been displaced by water-related disasters such as floods and rising water levels.

This project focuses on presenting information clearly and encouraging visitors to support affected communities through donations and humanitarian assistance.

---

## Project Purpose

The purpose of this website is to:

- create awareness about communities displaced by water disasters,
- encourage people to support humanitarian response,
- provide a simple and accessible donation information page,
- promote digital engagement for disaster support initiatives.

This project is intended for learning and social impact purposes.

---

## Key Features

- Home page with project information
- Donation information section
- Clear call-to-action for supporters
- Simple and responsive layout
- Clean and accessible design
